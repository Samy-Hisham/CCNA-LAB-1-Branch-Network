# CCNA Branch Network Lab

## Project Overview
This project is a Cisco Packet Tracer network simulation for a branch office network.

## Technologies & Concepts Used
- VLAN Configuration
- Trunk Links
- Inter-VLAN Routing
- DHCP Configuration
- Access Control Lists (ACL)
- Router and Switch Configuration

## Project Files
- CCNA-Lab.pkt : Cisco Packet Tracer project file
- topology.png : Network topology diagram
- screenshots : Configuration screenshots

## Objective
Design and configure a secure branch network using Cisco networking technologies.
